# Mindify

An AI-powered learning assistant that helps you study smarter.

## Features

- **Smart Summaries** — Automatically generate concise summaries from your notes
- **Flashcard Generation** — AI creates flashcards from any content
- **Adaptive Quizzes** — Test yourself with dynamically generated questions
- **Progress Tracking** — Monitor your learning over time

## Getting Started

```bash
git clone https://github.com/dariuspop/mindify
cd mindify
npm install
npm run dev
```

## Tech Stack

| Technology | Purpose |
|---|---|
| Next.js | Frontend framework |
| TypeScript | Type safety |
| TailwindCSS | Styling |
| PostgreSQL | Database |
| OpenAI API | AI features |
